# 📌 코디세이 AI 네이티브 과정 커뮤니티 이용 규칙

이 서버는 「코디세이 AI 네이티브 과정」 운영을 위한 공식 커뮤니케이션 채널입니다. 
모든 활동은 여러분이 서명한 참여 서약서·교육생 선서·비밀유지 의무 준수 서약서 및 Discord 이용약관을 따릅니다.


## 1. 존중과 배려

  ▶︎ 모든 구성원을 존중하고 상호 배려합니다.

  ▶︎ 차별·괴롭힘·비방·따돌림성 발언, 경쟁만을 부추기는 언행을 금합니다.

  ▶︎ 동료평가 시 재촉하거나 무시하는 발언을 하지 않습니다.

## 2. 건전한 채널 이용

  ▶︎ 부적절한 언어·이미지·도배(스팸)로 타인에게 불편을 주지 않습니다.

  ▶︎ 채널 주제에 맞게 글을 작성합니다.

## 3. ⚠️ 정보 보안

  ▶︎ 교육 자료, 소스코드·알고리즘, 운영진이 제공한 모든 문서·파일을 사전 동의 없이 외부에 유출·공유·복제하지 않습니다.

  ▶︎ 유출된 자료(GitHub 등)를 학습에 이용하지 않습니다.

  ▶︎ 유출 정황을 발견하면 즉시 운영진에게 신고합니다.

## 4. 공정한 참여

  ▶︎ 출입·접속 기록 조작, 대리 서명, 지원금 부정수급 등 일체의 부정행위를 하지 않습니다.

## 5. 운영 협조

  ▶︎ 운영진·퍼실리테이터의 안내에 따르며, 부정적 분위기를 선동·조성하지 않습니다.

## 6. 계정 및 닉네임

  ▶︎ 1인 1계정으로 참여하며 계정을 공유·양도하지 않습니다.

  ▶︎ 서버 닉네임은 실명으로 설정합니다.

## 7. 위반 시 조치

  ▶︎ 규칙 또는 서약 위반 시 경고 → 채널 이용 제한 → 서버 추방 → 교육 운영 규정에 따른 제재(과정 제외 등)가 단계적으로 적용될 수 있습니다.
